关联/hash数组

创建关联数组，必须先创建一个空数组，不能直接创建创建关联数组
原理: 接收一个字符串，并计算出尽量不重复的序号。
存入数据: hash算法接收一个字符串的下标名，将数据存储在序号对应的数组位置上
如何获取： 通过hash算法，算出和存入完全相同的序号，直接去序号位置查找元素。
有点： 查找极快
